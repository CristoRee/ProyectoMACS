User Manual — ProyectoMACS

Step-by-step guide to use the platform from the browser (without code references).

1. Getting Started

Access the application URL provided by your administrator. We recommend using an updated browser (Chrome, Edge, Firefox).

Registration and Access

Go to the main page
Click "Register" in the top menu
Fill in your personal information (name, email, phone)
Choose a secure password
Click "Register" to complete the process
Note: Your account will be activated by the administrator and assigned appropriate permissions.

2. Navigation and Interface

The system interface is intuitive and organized into several main sections:

  • Home: Main dashboard with quick access to key functions
  • My Profile: Manage your personal information and preferences
  • Tickets: View and manage repair requests
  • Chat: Communicate with technicians or clients
  • About Us: Information about BinaryTEC

3. Creating a Repair Request

To request device repair:

Click "Request Repair" on the home page
Select the device type from the list
Describe the problem in detail
Attach photos if necessary (optional)
Click "Submit Request"
Tip: Be as detailed as possible when describing the problem to help our technicians provide better service.

4. Request Tracking

You can track the status of your requests in "My Requests":

  • Pending: Request received, awaiting technician assignment
  • In Progress: Technician assigned and working on your device
  • Completed: Repair finished successfully
  • Cancelled: Request cancelled for various reasons

5. Real-time Chat

The chat system allows direct communication with assigned technicians:

Access "Active Chats" from the floating menu
Select the conversation related to your ticket
Type your message in the text field
Press Enter or click the send button
Important: Chat is available once a technician has been assigned to your request.

6. Profile Management

In your profile you can:

  • Update your personal information
  • Change your profile picture
  • Modify your password
  • Select interface language
  • Enable/disable dark mode

7. Tips and Recommendations

  • Keep your contact information up to date
  • Check your requests regularly for status updates
  • Use chat to clarify doubts with technicians
  • Take clear photos of damaged devices
  • Be specific when describing problems

8. Technical Support

If you need additional help:

  • Use the chat system for technical questions
  • Contact the administrator for account issues
  • Check this manual for step-by-step procedures
